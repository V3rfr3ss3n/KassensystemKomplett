# Kassensystem

Dieses Projekt ist die Startbasis für ein Java 21 + Maven + JavaFX Kassensystem.

## Starten

```bash
mvn clean javafx:run
```

## Struktur

- src/main/java/de/mmbbs/kassensystem/Main.java – Einstiegspunkt der JavaFX-Anwendung
- src/main/resources/styles.css – Platzhalter für späteres Styling
